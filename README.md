# Overview
WitnessLens AI is an AI-powered digital forensic and investigation assistant designed to analyze multiple
witness statements. It extracts key facts from PDF statements, compares testimonies, detects agreements
and contradictions, reconstructs event timelines, evaluates witness reliability, and generates investigation-ready
PDF reports.

# Features
* Upload witness statements in PDF format
* AI-based fact extraction using Google Gemini
* Named Entity Recognition (People, Locations, Time, Objects)
* Multi-witness comparison
* Agreement and contradiction detection
* Timeline reconstruction
* Witness reliability scoring
* Investigation report generation (PDF)
* Interactive dashboard for investigation analysis

# Workflow

1. Upload witness statements.
2. Extract structured facts using AI.
3. Compare multiple witness statements.
4. Detect agreements and contradictions.
5. Reconstruct the event timeline.
6. Calculate witness reliability.
7. Generate a complete investigation report.
